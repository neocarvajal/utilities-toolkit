A lightweight JavaScript utility library built from scratch to strengthen core programming fundamentals and software engineering skills.

This project includes reusable utility functions for handling strings, arrays, and objects, focusing on clean code, modular architecture, and real-world engineering practices.

## 🚀 Features

- String utilities (capitalize, slugify, truncate)
- Array utilities (unique, chunk, groupBy)
- Object utilities (deepClone, merge, pick)
- Modular ES Modules architecture
- Clean and maintainable code structure

## 🧠 Purpose

This project is part of a structured learning roadmap to master JavaScript core concepts, improve problem-solving skills, and build production-level coding habits.

It focuses on:
- JavaScript fundamentals
- Functional programming concepts
- Code organization and reusability
- Git best practices

## 🛠️ Tech Stack

- JavaScript (ES Modules)
- Node.js
- Git

## 📌 Goal

To build strong software engineering foundations by implementing utility functions from scratch without external dependencies.
